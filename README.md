# 떡국

1. 비비고 사골국물 육수를 준비한다.
2. 사골 국물에 물을 조금 추가하여 만두, 떡을 넣고 끓인다.
3. 끓는 동안 다른 재료를 준비한다. ( 다진마늘, 계란, 대파, 미원 )
4. 계란을 풀고 다진마늘과 대파를 섞는다.
5. 끓고 있는 냄비에 계란을 푼다.
6. 팔팔 끓이고, 미원을 넣어 간을 맞춘다.
